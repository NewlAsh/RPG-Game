# 🎮 Guided RPG Game

A simple browser-based guided RPG built using JavaScript.
This project focuses on learning core programming concepts through an interactive, choice-based game.

---

## 🚀 Features

* 🧭 Choice-based gameplay
* ⚔️ Basic combat system
* 🎒 Inventory management
* 📈 XP and health tracking
* 🔁 Restart and progression system

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

---

## 🎯 Purpose

This project was created as a beginner-friendly way to understand:

* JavaScript basics
* DOM manipulation
* Game logic design
